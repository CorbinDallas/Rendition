Rendition.Commerce This class contains static methods used indirectly by the JSON responder and adminResponder and internally by other methods. The classes Rendition.merchant and Rendition.Admin use System.Reflection to expose Their static methods to the JSON responder and adminResponder and wrap some of the methods in this class. Class Rendition.Admin contains all the dangerous CRUD methods that should only be avaliable to administrators while Rendition.merchant contains only methods that should be avaliable to anonomyous internet users. Rendition.Commerce (this class) contains methods that both Rendition.Admin and Rendition.merchant wrap to make these methods accessable to the reflection methods used in the JSON response mapper JSONToMethod
