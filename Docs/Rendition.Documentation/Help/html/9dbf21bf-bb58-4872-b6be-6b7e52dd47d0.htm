How to create a basic web site using Rendition E-Commerce library.

If you're not familar with editing ASP.NET Web Forms a great set of tutorials can be found at http://www.w3schools.com/aspnet/.

All examples use C#, HTML 5, jQuery 1.7 and CSS 3. A good understanding of ASP.NET Web Forms is a good idea before you get started using Rendition.

default.aspx

Default.aspx is the default page that appears when a user visits your site and does not define a document to view. (E.g.:http://www.mysite.com/)

@ commands

The following commands take place at the very top of the page.

The following command defines this page's master page file.

Copy 
<%@ Page MasterPageFile="~/header.master" Language="C#" %>
The next command includes a reference to Rendition.dll
Copy 
<%@ Import Namespace="Rendition" %>

asp:ContentPlaceHolder and asp:Content Tags

The remainder of the page consists of <asp:ContentPlaceHolder> sections. Content placed within the <asp:ContentPlaceHolder> section will be inserted into the master page file, header.master in this case, and then sent to the client's browser as default.aspx.

Note

Web Forms (.aspx files) are not required to have a master page file.

Content you want to inject into the body section of the html document should be placed in the body <asp:ContentPlaceHolder>. For example:

Copy 
<asp:Content ID="body" ContentPlaceHolderID="body" Runat="Server">
    Lorem ipsum dolor sit amet, consectetuer adipiscing elit, sed diam nonummy nibh euismod tincidunt ut 
    laoreet dolore magna aliquam erat volutpat. Ut wisi enim ad minim veniam, quis nostrud exerci tation ullamcorper suscipit lobortis
    nisl ut aliquip ex ea commodo consequat.
</asp:Content>

You can inject code into any of the <asp:ContentPlaceHolder>s on the master page (header.master). To select which <asp:ContentPlaceHolder> to inject into match the id attribute of the <asp:ContentPlaceHolder> with the attribute "ContentPlaceHolderID" in the <asp:Content> tag. For example, if you want to inject code into the "footer" section, you would type the following:

Copy 
<asp:Content ID="footer" ContentPlaceHolderID="footer" Runat="Server">
    Duis autem vel eum iriure dolor in hendrerit in vulputate velit esse molestie consequat,
    vel illum dolore eu feugiat nulla facilisis at vero eros et accumsan et iusto odio dignissim qui blandit praesent luptatum zzril 
    delenit augue duis dolore te feugait nulla facilisi.
</asp:Content>

Note

You can create as many <asp:ContentPlaceHolder> and <asp:Content> as you want and the id's can be anything you dream up, these are just suggestions.
