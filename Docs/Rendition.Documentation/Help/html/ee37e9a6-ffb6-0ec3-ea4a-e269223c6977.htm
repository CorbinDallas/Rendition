Rendition.UI.FileManager Methods

Name

Type

Description

Cp (Array sourcePaths, Array targetPath, Function callbackProcedure, Object pane)

Object

Copies one or more file or directory and subdirectories to another directory. Similar to UNIX "cp" or windows "xcopy /y /e".

edit (String path, Boolean newFile, Function saveCallbackProcedure)

Object

Edit's the file on the server by loading the file text into an instance of Rendition.UI.CodeEditor.

ls (Array paths, Function callbackProcedure, Object pane)

Array

Gets a list of files and directories in a directory. Similar to UNIX "ls" or windows "dir".

mkdir (String Path, Function callbackProcedure, Object pane)

Object

Creates a directory.

mv (Array sourcePaths, Array targetPath, Function callbackProcedure, Object pane)

Object

Moves one or more file to another directory. "mv" or windows "move /y".

rm (Array paths, Function callbackProcedure, Object pane)

Object

Recursivly removes a file or directory. Similar to UNIX "rm -r" or windows "deltree /y".

rn (String sourcePath, String targetPath, Function callbackProcedure, Object pane)

Object

Renames a directory or file. Same as UNIX "mv" or windows "rename".
