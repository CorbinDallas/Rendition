Rendition.UI.TreeView Events

Name

Description

onclose (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Occurs when a tree node is toggled closed to hide its child nodes.

ondrag (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Occurs when a node is being moved.

ondragover (Object e, TreeView treeView, Object dragSource, Object dragTarget)

Occurs when a node is being moved over another node.

ondrop (Object e, TreeView treeView, Object dragSource, Object dragTarget)

Occurs when a node is being moved.

onlabelclick (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Occurs when a tree node's label is clicked on.

onlabelcontextmenu (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Occurs when a tree node's label's context menu is triggered. This usualy due to a right mouse button click (or Command+click for Mac users).

Note

It is important you use the method e.preventDefault when subscribing to this event. Failure to do so will cause the system's context menu to appear.

onlabeledit (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Not implemented. Occurs when a tree node's label is being edited.

onlabelendedit (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Not implemented. Occurs when a tree node's label is exiting edit mode. treeView.preventDefault will prevent the exit from edit mode.

onlabelkeydown (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Not implemented. Occurs when a tree node's label's edit input keydownevent occurs.

onlabelkeypress (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Not implemented. Occurs when a tree node's label's edit input keypress occurs.

onlabelkeyup (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Not implemented. Occurs when a tree node's label's edit input keyup occurs.

onlabelstartedit (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Not implemented. Occurs when a tree node's label is entering edit mode. treeView.preventDefault will prevent entering edit mode.

onopen (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Occurs when a tree node is toggled open to reveal its child nodes.

onrefresh (Object e, TreeView treeView)

Occurs when the tree is being reinitalized from the original parameters.

onresize (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Occurs when the treeview is being resized.

onstartdrag (Object e, TreeView treeView, DHTMLElement node, DHTMLElement labelText, TreeNode treeNode, DHTMLElement parentNode)

Occurs when a node has been started to drag.

Tip

There are two ways to add event listeners.

1. You can attach to events as a parameter when you create the object.

CopyJavaScript
var foo = new object({
    event:function(signature){
        /* event procedure */
    }
});
2. Or you can add the event after the object has been instantiated.
CopyJavaScript
var foo = new object();
foo.addEventListener('event',function(signature){
    /* event procedure */
}, false);
