Class Rendition.UI.AutoComplete

Creates a DHTML based autocomplete. This class implements Rendition.UI.ContextMenu to create a drop down menu when [args.autoComplete] is true. It also provides methods for validating user input against Regular Expresions and database records. Most all validation in Rendition is done using this class.

Examples

Bind autocomplete to an input and prevent invalid input.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var input = document.createElement('input');
var submit = document.createElement('button');
var bar = Rendition.UI.AutoComplete({
    mustMatchPattern: /^.+$/,
    patternMismatchMessage: 'This field cannot be blank.',
    patternMismatchTitle: 'Invalid Value.',
    input: input
});
submit.innerHTML = 'Check validation';
foo.content.appendChild(input);
foo.content.appendChild(submit);
submit.onclick = function () {
    alert(bar.isValid());
}
});

Bind autocomplete to an input. Require the value to be present in the result of the query matching the object + suffix and require the value to match a pattern.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var input = document.createElement('input');
var submit = document.createElement('button');
var bar = Rendition.UI.AutoComplete({
    mustMatchPattern: /[0-9]+|^$/i,
    patternMismatchMessage: 'This field can only contain the numbers 0-9.',
    patternMismatchTitle: 'Invalid value.',
    objectName: 'shortUserList',
    suffix: "where userId like '%<value>%' or handle like '%<value>%' or email like '%<value>%'",
    mustMatchRecord: true,
    matchSuffix: "where userId = '<value>'",
    autoComplete: true,
    optionDisplayValue: '<column0> <column1>',
    optionValue: '<column0>',
    input: input
});
submit.innerHTML = 'Check validation';
foo.content.appendChild(input);
foo.content.appendChild(submit);
submit.onclick = function () {
    alert(bar.isValid());
}

The suffix (where clause) used check if the value is an acceptable match to a value in the database.

args.selectURL

Function

The URL the autocomplete class uses. The function's signature is function(objectName, suffix, from, to, selectedRows, checksum, outputType, searchSuffix, includeSchema, orderBy, orderDirection, aggregateColumns). The result will be sent to the Rendition.UI.autocompleteCallback procedure which must return a DataSet. The default function calls the C# Rendition.Admin.DataSet method.

args.callback

Function

This function processes the data returned from the server when the Rendition.UI.autocompleteURL is sent. The type returned by this function must be a DataSet. The signature of the function is Function(XMLHttpRequest e, AutoComplete autoComplete). The default function uses the C# Rendition.Admin.DataSet method.
