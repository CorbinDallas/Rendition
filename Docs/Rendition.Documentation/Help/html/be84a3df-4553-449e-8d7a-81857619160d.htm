Site Template Design

This topic and subtopics in this section will help you design a front facing retail web site from scratch, explaining the content of each page and how to implement the rendition namespace to create an interactive web site.

The pages are listed in alphabetical order. Becuase Rendition is an HTTPModule you don't need to call on it with special commands to make it run. So long as Rendition is refrenced in your web.config it will run. See Installation for more information.

Special URLs

The following Urls are special. They are intercepted by Rendition and replaced by either speical html pages or respond to JSON commands issued from your JavaScript commands on your front end site. These URLs are defaults, you can change the defaults by modifying your web.config. Becuase these URLs are used by Rendition, you cannot create URLs with the same name. See Installation for more information.

  • /responder
  • /admin
  • /admin/responder
