Class Rendition.UI.ContextMenu

Creates a DHTML based context or dropdown menu. This class requires an array of MenuOption Objects. The context menu can operate in four modes.

Mode

Description

mouse

Causes the context menu to appear where the mouse has just clicked.

RenditionContextMenu

Causes the menu to appear as a child of another contextMenu.

RenditionAutoComplete

Causes the menu to appear below the input to which it was bound.

RenditionMenuBar

Causes the menu to below the element it was bound to.

This class registers itself with the Desktop on which it occured and makes sure no other context menus are visible.

Examples

Create multiple options and bind to the context menu event.

CopyJavaScript
                    // Create a dialog //
var foo = Rendition.UI.Dialog();
// Create a button //
var bar = document.createElement('button');
bar.innerHTML = 'Right Click On Me.';
foo.content.appendChild(bar);
// Create an array to add the menuOption objects to //
var options = [];
options.push(new Rendition.UI.MenuOption({
    text:'Tres',
    mousedown:function (e, menu) {
        alert('Facer possim assum.');
    }
}));
options.push(new Rendition.UI.MenuOption({
    text: 'Quattuor',
    hasChildren: true,
    mouseover: function (e, menu) {
        // Create a child menu //
        var childOptions = [];
        childOptions.push(new Rendition.UI.MenuOption({
            text: 'Quattuor',
            mousedown: function (e) {
                alert('Eodem modo typi.');
            }
        }));
        var menu = Rendition.UI.ContextMenu(e, {
            elements: childOptions,
            caller: this,
            type: 'RenditionContextMenu'
        });
        // you need to run this to make sure all menus that are _not_ a ancestor of this menu get closed //
        Rendition.UI.closeContextMenus(menu);
    }
}));
// Create the context menu event on the button element //
bar.oncontextmenu = function (e) {
    var menu = Rendition.UI.ContextMenu(e, {
        elements: options,
        caller: bar,
        type: 'bar'
    });
    e.preventDefault();
}

Constructor Parameters

Name

Type

Description

e

Object

The browser event object.

args

Object

Parameters for the widget.

args.elements

Array

Array of MenuOption objects that represent the values in this menu.

args.scrollRate

Integer

The rate the context menu scrolls when the mouse is over the scroll icon.

args.elementStyle

Object

The style of the menuOption object that belongs to this menu. An instance of MenuOptionStyle.

args.style

Object

The style of the menu. An instance of ContextMenuStyle.

args.caller

DHTMLElement

The DHTML element associated with this menu.

args.callerType

String

The type of caller. Can be any one of RenditionDatagridStatus, RenditionContextMenu, RenditionAutoComplete, RenditionMenuBar, mouse or undefined.
