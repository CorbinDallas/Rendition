Class Rendition.UI.TreeView

Creates a DHTML based tree view. The tree view uses the natural hierarchial nature of the DOM to create a tree. Tree nodes can be a TreeNode Object, a function or a URL and callback function.

When a TreeNode object is added to a treeView it will get a few extra properties from the treeView and will end up looking like this

CopyJavaScript
{
    
    treeNode:{
        text: String,
        value: Object,
        childNodes: Array,
        URL: String,
        callbackArguments: Array,
        callbackProcedure: Function
    },
    parentNode: DHTMLElement,
    element: DHTMLElement,
    treeNodeId: Integer,
    parentTreeNode: Object
}

If the property URL is defined in a treeNode the URL will be requested when the node's open event occurs. when the request is complete the Array [callbackArguments] will be prefixed to the event signature: (json, tree, siblingNodes, node). The [callbackProcedure] will then be applied to the response within the context of the treeView. The function you provide should create treeNodes when it is fired. An example of this can be found in the list of examples.

The treeNode object above can be found by selecting a DHTML Element in the treeView and looking at its treeNode property, by using the [f6625a7b-e83f-4e49-2823-564958c8aacd] method to get the node by its treeNodeId or by searching through the [3e52e04c-86e5-3241-1d6a-aea297b85853] array for the desired node.

Examples

Create a new dialog. Create a treeView and attach it to the dialog. Add a two tree nodes.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var bar = Rendition.UI.TreeView({
    parentNode: foo.content
});
var unus = bar.add({
    text: 'Duis autem vel.',
    value: 'unus',
    childNodes: []
});
var duo = unus.add({
    text: 'Minim veniam.',
    value: 'duo',
    childNodes: []
});

Create a new dialog. Create a treeView and attach it to the dialog. Use a URL and callback procedure as a child node. Bind an event to the label's context menu.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var bar = Rendition.UI.TreeView({
    parentNode: foo.content,
    dragMode: 0,
    includeRoot: true,
    labelcontextmenu: function (e, treeView, node, labelText, treeNode, parentNode) {
        var options = [];
        options[0] = Rendition.UI.MenuOption();
        options[0].text = 'Remove Node';
        options[0].addEventListener('mousedown', function () {
            node.remove();
        }, false);
        options[1] = Rendition.UI.MenuOption();
        options[1].text = 'Show Node Value';
        options[1].addEventListener('mousedown', function () {
            alert(treeNode.value);
        }, false);
        new Rendition.UI.ContextMenu(e, {
            elements: options,
            caller: this,
            type: 'mouse'
        });
        e.preventDefault();
        treeView.preventDefault();
    },
    rootNode: {
        text: 'Unus.',
        value: 'unus',
        childNodes: [
            {
                text: 'Duis vel.',
                url: Rendition.UI.clientServerSyncURI + Rendition.UI.responderKeyName + '1=' + JSON.stringify(['GetSqlArray',
                [{ commandText: 'select userId from users with (nolock)'}]]),
                callbackArguments: [],
                callbackProcedure: function (json, tree, siblingNodes, node) {
                    var a = json.method1.GetSqlArray;
                    var l = a.length;
                    for (var x = 0; l > x; x++) {
                        var newNode = {
                            text: a[x][0],
                            value: a[x][0]
                        }
                        if (!node.containsNode(newNode)) {
                            node.add(newNode);
                        }
                    }
                    return false;
                }
            }
        ]
    }
});

Check the result of drag and drop's new node ordering.

CopyJavaScript
                    //Create a new dialog.
var foo = Rendition.UI.Dialog();
// Create a treeView and attach it to the dialog.
// Use a URL and callback procedure as a child node.
// Check the order of the nodes after the user has reordered the nodes.
var bar = Rendition.UI.TreeView({
    parentNode: foo.content,
    dragMode: 0,
    rootNode: {
        text: 'Unus.',
        value: 'unus',
        childNodes: [
            {
                text: 'Duis vel.',
                url: Rendition.UI.clientServerSyncURI + Rendition.UI.responderKeyName + '1=' + JSON.stringify(['GetSqlArray',
                [{ commandText: 'select userId from users with (nolock)'}]]),
                callbackArguments: [],
                callbackProcedure: function (json, tree, siblingNodes, node) {
                    var a = json.method1.GetSqlArray;
                    var l = a.length;
                    for (var x = 0; l > x; x++) {
                        var newNode = {
                            text: a[x][0],
                            value: a[x][0]
                        }
                        if (!node.containsNode(newNode)) {
                            node.add(newNode);
                        }
                    }
                    return false;
                }
            }
        ]
    },
    drop: function (e, tree, dragSource, dragTarget) {
        var p = dragSource.parentNode;
        var l = p.childNodes.length;
        var order = [];
        for (var x = 0; l > x; x++) {
            order.push({
                value: p.childNodes[x].treeNode.value,
                order: x
            });
        }
        alert(JSON.stringify(order));
    }
});

Constructor Parameters

Name

Type

Description

args

Object

Parameters for the treeView widget.

args.parentNode

DHTMLElement

The DHTML element to append the widget to.

args.rootNode

Object

The inital treeNode to add to the treeView.

args.dragMode

Integer

The mode that node dragging works in. undefined = No dragging allowed. 0 = Stay within the same parent. 1 = Move from any node to any other node in the same TreeView instance.

args.dragMode

Object

The mode that node dragging works in.

args.includeRoot

Boolean

Include the root tree node. All the nodes are the decendant of a single node. When this parameter is true this root node will be visible.

Tip

There are two ways to add event listeners.

1. You can attach to events as a parameter when you create the object.

CopyJavaScript
var foo = new object({
    event:function(signature){
        /* event procedure */
    }
});
2. Or you can add the event after the object has been instantiated.
CopyJavaScript
var foo = new object();
foo.addEventListener('event',function(signature){
    /* event procedure */
}, false);
