Class Rendition.UI.Grid

Creates a DHTML based data grid.

Grid can synchronously or asynchronously read from a recordset you provide at runtime or it can read directly from a MS-SQL Database. Grid provides default functionality like edit, search and export.

Grid also provides events you can subscribe to so you can extend the functionality of the grid. When used on a table or view that implements the VerCol timestamp column records are tracked for their edit state. This means if someone edits the record that you're editing you'll be warned that the data on the server has changed.

Grid can also work as a recordset fetching tool. You can use grid to search through a large table and return relivent records, create, update, delete and read records synchronously or asynchronously.

Full pramatization and strong typing for MS SQL databases supported. By default grid streams in data as you scroll. Only when you have scrolled past the page does the data stream in from the server via asynchronous requests to the server that return JSON data.

Examples

Create a new dialog. Create a new grid and attach it to the dialog.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var bar = Rendition.UI.Grid({
    objectName: 'items',
    parentNode: foo.content
});

Create a new grid and attaches a cellclick event to it that sets the title of the foo dialog to the value of the cell clicked.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var bar = Rendition.UI.Grid({
    objectName: 'items',
    parentNode: foo.content,
    cellclick: function (e, grid, element, row, column, selection, data, header) {
        foo.title(grid.getRecord(row, column));
        return;
    }
});

Create a dialog, add a grid that uses the generic editor instead of the inline editor.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var bar = Rendition.UI.Grid({
    editMode:3,
    genericEditor: true,
    objectName: 'items',
    parentNode: foo.content
});

Creates a grid for use as a recordset handling class. After the grid loads a record from the recordset is displayed.

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var foo = Rendition.UI.Grid({
    objectName: 'users',
    afterloadcallback: function(e, grid, element, row, column, selection, data, header){
        alert(bar.getRecord(1,'handle'));
    }
});

onviewportcontextmenu (Object e, Object grid, DHTMLElement element, Integer row, Integer column, Array selection, Array data, Object header)

Occurs when a context menu is called anywhere on the grid.

onviewportdblclick (Object e, Object grid, DHTMLElement element, Integer row, Integer column, Array selection, Array data, Object header)

Occurs when a mouse button is double clicked anywhere on the grid.

onviewportmousedown (Object e, Object grid, DHTMLElement element, Integer row, Integer column, Array selection, Array data, Object header)

Occurs when a mouse button is depressed anywhere on the grid.

onviewportmousemove (Object e, Object grid, DHTMLElement element, Integer row, Integer column, Array selection, Array data, Object header)

Occurs when the mouse moves within the grid.

onviewportmouseout (Object e, Object grid, DHTMLElement element, Integer row, Integer column, Array selection, Array data, Object header)

Occurs when the mouse moves out of the grid.

onviewportmouseover (Object e, Object grid, DHTMLElement element, Integer row, Integer column, Array selection, Array data, Object header)

Occurs when the mouse moves out of a header cell.

onviewportmouseup (Object e, Object grid, DHTMLElement element, Integer row, Integer column, Array selection, Array data, Object header)

Occurs when a mouse button is released anywhere on the grid.

Tip

There are two ways to add event listeners.

1. You can attach to events as a parameter when you create the object.

CopyJavaScript
var foo = new object({
    event:function(signature){
        /* event procedure */
    }
});
2. Or you can add the event after the object has been instantiated.
CopyJavaScript
var foo = new object();
foo.addEventListener('event',function(signature){
    /* event procedure */
}, false);
