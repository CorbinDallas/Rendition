How to create a basic web site using Rendition E-Commerce library.

If you're not familar with editing ASP.NET Web Forms a great set of tutorials can be found at http://www.w3schools.com/aspnet/.

All examples use C#, HTML 5, jQuery 1.7 and CSS 3. A good understanding of ASP.NET Web Forms is a good idea before you get started using Rendition.

detail.aspx

Detail.aspx is the page that shows the detail of an item on your site. The example below looks for a URN key called "item".

Note

To find out what a URN key is, read the topic All About URLs

To access the key, you need to use the built in Request object. In the example below, the Request object is used to fetch a key value from the URI and use that value to lookup an item in the Rendition item database.
Note

It would help to have an item in your database, so if you haven't already created an item, now would be a good time to do so. See Item Editor for more info on creating items.
