Site Management Applications

Email

See what emails are going in and out of your site, and send emails to one or more users.

Imaging Templates

Create, enable, disable and execute templates on images uploaded to the site. Add effects like resize, contrast, text overlay and more.

Import

Import items, users, shipping services and more from CSV or XLS files.

Logon

Logon, logoff and a password changing utility.

Scripting

Create scripts that attach to event handlers on the site. Such as when the site starts, when a user connects, when an email is sent, image rendered and a lot more.

Search

Search your site's database, create complex conditions and export the results to a CSV/XML or HTML file or use the results to get at information in your site.

Site Configuration

Alter all site defaults. Create sizes and swatches, divisions. Check the startup, process, plugins, timers and more.

Site Statistics

See who's visiting your site, and what they're looking at.

URL Rewrite

Rewrite URLs, so customers see more human readable URLs, and fix old broken links by pointing them to new locations.

User Interface: Grid

Learn the basics of how to use the grid widget.

User Interface: Date Chart

Learn how to manipulate date driven charts.

User Interface: Dialog

Learn how to manipulate the basic dialog window widget.

User Interface: Cutter

Learn how to manipulate the basic cutter bar widget.
