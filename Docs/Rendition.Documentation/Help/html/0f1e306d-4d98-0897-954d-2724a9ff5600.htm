Class Rendition.UI.NumericUpDown

Creates a DHTML based numeric up down widget that binds itself to a DHTML input element. You can specifiy the increment, min, max repeat rate and more.

Member Of

UI

Source

c:\Rendition\Rendition.Core\js\admin\UI\numericupdown.js

Examples

Bind a numericUpDown widget to an input

CopyJavaScript
                    var foo = Rendition.UI.Dialog();
var bar = document.createElement('input');
foo.content.appendChild(bar);
Rendition.UI.NumericUpDown({
    input: bar
});
