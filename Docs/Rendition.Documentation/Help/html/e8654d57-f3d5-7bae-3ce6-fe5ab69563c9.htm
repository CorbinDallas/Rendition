Class Rendition.UI.FileManager

Creates a DHTML based file manager that allows you to upload/download/copy etc. with the server and remote FTP or UNC paths. Can also function as a 'select a file(s)' or 'select a folder(s)' dialog.

Member Of

UI

Source

c:\Rendition\Rendition.Core\js\admin\UI\fileManager.js

Examples

Create a file manager

CopyJavaScript
Rendition.UI.FileManager();

Create a file manager that asks the user to select a file, then displays the selection in an alert

CopyJavaScript
                    var foo = Rendition.UI.FileManager({
        selectFile: true,
        selectCallback: function (selectedFilePath, fileMan) {
            alert(selectedFilePath);
            return;
        }
});

Constructor Parameters

Name

Type

Description

args.parentNode

DHTMLElement

DHTML element to append to. If left undefined a dialog will be created.

args.selectFile

Boolean

When true, turns the file FileManager into a 'select a file' dialog that works with the args.selectCallback parameter to return a selected file.

args.selectFolder

Boolean

When true, turns the file FileManager into a 'select a folder' dialog that works with the args.selectCallback parameter to return a selected folder.

args.selectFiles

Boolean

When true, turns the file FileManager into a 'select a file' dialog that works with the args.selectCallback parameter to return a selected files.

args.selectFolders

Boolean

When true, turns the file FileManager into a 'select a folder' dialog that works with the args.selectCallback parameter to return a selected folders.

args.selectCallback

Function

A function to execute when the user comfirms their selection. Function signature:

CopyJavaScript
function(selectedFilesOrFoldersPath, fileManager){
        /// procedure code ///
    }
