The expander control arrow that points right or down depending on the toggled state.

id

Unique id of this widget.

Integer

Index of this groupBox in the Rendition.UI.groupBoxes array

inner

DHTMLElement

The inner most DIV, this DIV holds the content of the groupBox. You should use the groupBox.appendChild method to append child element. Don't use this node directly or risk breaking some functionality.

outer

DHTMLElement

The DIV that holds everything in the group box.

tspan

DHTMLElement

The DIV that holds the title text.
