Rendition.UI.TabBarTab Events

Name

Description

onafterloadcontent (Object tab, Object tabBar, Object content)

Occurs after the remote procdure call has returned.

onclick (Object tab, Object tabBar, Object content)

Occurs when the user clicks on the TabBarTab.

onload (Object tab, Object tabBar, Object content)

Occurs when the TabBarTab is finished loading and is ready to accept content.

onmousedown (Object tab, Object tabBar, Object content)

Occurs when the mouse button is pressed down on the TabBarTab.

onmouseout (Object tab, Object tabBar, Object content)

Occurs when the mouse moves out of the TabBarTab.

onmouseover (Object tab, Object tabBar, Object content)

Occurs when the mouse moves over of the TabBarTab.

onmouseup (Object tab, Object tabBar, Object content)

Occurs when the mouse button is released on the TabBarTab.

onresize (Object tab, Object tabBar, Object content)

Occurs when the parent TabBar is resized.

Tip

There are two ways to add event listeners.

1. You can attach to events as a parameter when you create the object.

CopyJavaScript
var foo = new object({
    event:function(signature){
        /* event procedure */
    }
});
2. Or you can add the event after the object has been instantiated.
CopyJavaScript
var foo = new object();
foo.addEventListener('event',function(signature){
    /* event procedure */
}, false);
