[This is preliminary documentation and is subject to change.]

The List type exposes the following members.

Properties

  Name Description
Category
The category loaded.
CurrentList
Gets the current list stored for this HttpContext or creates a list with default options if one is not yet stored.
FilterTable
A list of filters for this list in a table with the class filtertable.
Menu
The menu this list will display. When no entry is found the entry named Default will be used.
Meta
The meta title, descriptions and keywords. When no meta entry is found the entry named Default will be used.
Options
The options in use for this listing.
PriceFilterTable
A list of price filters for this list.
PriceRanges
List of price range filters
RemoveFilterTable
Gets the remove filter table.
Section
The site section of this list. When no section is found matching this list the section named Default will be used.
