Class Rendition.UI.ResizeableBox

Creates a DHTML based resizeable box with a resizer in the lower right corner. Ever want to have a DHTML DIV element that can be resized by the user? This widget doesn't work correctly.

Member Of

UI

Source

c:\Rendition\Rendition.Core\js\admin\UI\resizeableBox.js

Examples

Create a resizeable box

CopyJavaScript
                    var bar = Rendition.UI.ResizeableBox({
    resize: function () {
        bar.box.style.background = '#'+Rendition.UI.getFixedWidthRandomInt(10,6,16);
    }
});

bar.box.style.background = 'teal';

document.body.appendChild(bar.box);
